# Research

Papers, posts, and projects worth tracking. Add new entries as they're
discovered — the Knowledge Keeper pattern applies here too.

## Papers

- [Generative Agents: Interactive Simulacra of Human Behavior](https://arxiv.org/abs/2304.03442)
  (Park et al., 2023) — foundational work on memory stream architecture
  for believable agents. Introduced reflection, importance scoring, and
  retrieval by recency × relevance × importance.

- [MemoryBank: Enhancing Large Language Models with Long-Term Memory](https://arxiv.org/abs/2305.10250)
  (Zhong et al., 2023) — dynamic memory mechanism with forgetting curve
  inspired by Ebbinghaus. Memory updates based on access patterns.

- [Cognitive Architectures for Language Agents](https://arxiv.org/abs/2309.02427)
  (Sumers et al., 2023) — survey of cognitive science-inspired agent
  architectures. Connects working memory, episodic memory, and procedural
  memory to LLM agent design.

## Projects

- [memvid](https://github.com/memvid/memvid) — Rust. Single-file `.mv2`
  persistent memory with BM25 + HNSW vector search.
- [Mengram](https://github.com/mengram/mengram) — Three-tier cognitive
  memory with procedural evolution.
- [Basic Memory](https://github.com/basicmachines-co/basic-memory) —
  Knowledge graph with typed observations and relations.
- [Zep](https://github.com/getzep/zep) — Long-term memory for AI
  assistants. Entity extraction, temporal awareness, session summaries.
- [Mem0](https://github.com/mem0ai/mem0) — Memory layer for LLM
  applications. Hybrid search, user-level memory, graph memory.

## Posts & Talks

- [Compound Engineering](https://every.to/guides/compound-engineering) —
  AI-native dev methodology where each work cycle makes future cycles
  easier. Directly relevant to knowledge compounding.
- [Letta (MemGPT)](https://github.com/letta-ai/letta) — OS-inspired
  memory management for LLMs. Virtual context management with memory
  tiers.
